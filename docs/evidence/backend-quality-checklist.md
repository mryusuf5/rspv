# Backend quality checklist

Voor wijzigingen aan de backend controleer ik:

- Werkt registratie/login nog?
- Werkt PDF-upload nog?
- Werkt EPUB-upload nog?
- Worden pagina's goed opgeslagen?
- Wordt reading progress correct opgeslagen?
- Zijn foutmeldingen duidelijk?
- Zijn gegenereerde testbestanden niet meegenomen in Git?
- Zijn wijzigingen kort beschreven in commitberichten?
